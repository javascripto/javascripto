# 👨🏻‍💻 Yuri Alves

**`Software Engineer | Frontend & Fullstack Developer`**

Me chamo Yuri Alves, sou engenheiro de software com experiência em desenvolvimento **frontend e fullstack**, atuando principalmente com aplicações web escaláveis, integrações entre sistemas, produtos digitais e plataformas para o mercado financeiro.

Tenho experiência com **React, TypeScript, Next.js, Node.js, NestJS, Kotlin, Go, Java, Python** e outras tecnologias. Ao longo da minha trajetória, trabalhei em projetos envolvendo **fintechs, crédito, financiamento de veículos, antecipação de FGTS, empréstimos pessoais, BNPL, backoffices, APIs, microserviços, integrações, automações e ferramentas com IA**.

Gosto de construir software com foco em qualidade, clareza, arquitetura, manutenibilidade e impacto real para o negócio.
